# sequenceTask

自动识别屏幕内容并操作按脚本执行操作

依赖
```sh
pip install pyperclip pyautogui opencv-python nuitka pywin32
```

在程序所在目录新建一个文件夹，则为一个自动执行项目，在文件夹里新建 `sequence.csv` 作为操作脚本。并放置操作所需的目标识别图像文件。

## sequence.csv 说明

    每行是一个操作。
    第一个参数是执行前等待秒数。
    第二个参数是操作类型。
    第三个参数是操作类型所需参数，如果该操作类型需要识别定位目标，则应该写所需图像文件名称。
    第四个参数是最大等待时间(秒)，在需要图像识别的操作类型，最多会等带目标多久，超时则直接中断操作，不再执行后续操作。
